比特经济的到来

Network Address Translation (NAT)

NAT 模式使用了 NAT 服务来给虚拟网络提供网络连接。

这种模式下，虚拟机能访问外部网络，外部无法直接连接到内部网络，除非使用端口映射 port forwarding。
